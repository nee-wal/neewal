Neewal is a modern, minimal screen recorder desktop app. It features a sleek, dark-themed UI inspired by premium design aesthetics.

## Tech Stack

- **Framework**: [Electron](https://www.electronjs.org/)
- **Frontend**: [React](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) (v4)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Backend / Auth**: [Supabase](https://supabase.com/)

## Features

- **Modern UI**: Clean, dark interface with teal accents (`#3FB6A8`).
- **Recording Modes**: Screen, Window, and Region selection.
- **Audio Control**: Toggle microphone and system audio capture.
- **Format Selection**: Support for MP4, WebM, MKV, and GIF.
- **Settings**: Customizable save directory, frame rate, and cursor visibility.

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the app in development mode:

```bash
npm run dev
```

This will run the React dev server and launch the Electron window.

### Build

Build the project for production:

```bash
npm run build
```
